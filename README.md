A little party game made in 2 days for Global Game Jam 2026.  Given a prompt, build the perfect face for the occasion while weathering the game's onslaught of negativity.

Best Face Forward uses Unity's peer-to-peer networking solution to support online multiplayer, and leverages a state machine with a single update loop to run the game and prevent race conditions.  You can check it out at https://irateburritos.itch.io/best-face-forward.
